开始使用EFH5

拷贝项目

想要使用EFH5，你需要先将最基础的EFH5拷贝到你要用的地方。

最基本的EFH5包含以下文件：


index.html
regular.css

当然，这只是最基本的东西，就像是毛坯房一样。一般来说，EFH5还会有像icon.png这样的图片以及regular.js和jquery-3.7.1.min.js两个JavaScript文件。

获取途径

你可以直接从Github等平台将EFH5的Releases下载到本地。

也可以在本地建立上述文件后将内容复制进去。

当然，如果你会，也可以使用Git等工具。但是会用Git的人真的用得到这个项目吗？

目录结构

最终，你的EFH5项目目录应该长这个样子：


EFH5
├─ index.html
├─ regular.css
├─ regular.js//可能有
├─ jquery-3.7.1.min.js//可能有
└─ media
   └─ icon.png

添加部件

在完成上一步之后，你的EFH5项目应该还只有个简陋的页面。

接下来，可以去填写一些你的内容了。

下面，用文本编辑器打开index.html来开始内容的编辑。（这里推荐使用Sublime Text或Microsoft Visual Studio Code。当然，你想用记事本也是可以的。）

section的使用

EFH5页面的内容部分用<section></section>标签分成了多个部分，内容放在了section内。接下来，可以在<body></body>标签内插入<section></section>标签来创建一个section。

完成之后，body部分应该长这个样子：


<body>
<div class="main-flex">
<div class="main-box">
	<section>
	</section>
</div>
</div>
</body>

你可能会发现，里面除了<body>和<section>标签之外，还多了两个<div>标签，并且它们还带着class=""这样的内容。现在你可以不用管它，这是以后会用到的工具。

添加标题

接下来，你可以给这个section的内容添加一个标题了。

标题用到的标签是<h1></h1>。当然，数字可以更改为其它值，用来表示标题的不同层级。

现在，试着在section中放入一个内容为“首页”的标题。完成之后应该是这个样子：


<body>
<div class="main-flex">
<div class="main-box">
	<section>
		<h1>首页</h1>
	</section>
</div>
</div>
</body>
